### Coming Immediately Post-Beta: Message Intelligence Foundation

**Facts / Opinions / Emotions Classification**

Before we can track sentiment, we need to understand what we're reading. Our AI classifies every message:

- **Facts** - Objective events (verifiable: "Project X launched mainnet")
- **Opinions** - Reasoned analysis (subjective but argued: "I think X will pump because...")
- **Emotions** - Pure sentiment (gut reactions: "X is going to moon üöÄ")

**Why this matters:** You can't do sentiment analysis on facts. A mainnet launch is neutral - it's what people SAY about it that's bullish or bearish. This classification is the foundation for everything that comes next.

**Status:** In development now. Launching right after beta.

## üöÄ Phase 1: Enhanced Intelligence (Q2-Q3 2026)

### Focus: From categories to projects, from reading to tracking, cross-platform expansion

### Key Features

#### Twitter Integration (Q2 2026)
**Expand beyond Telegram - this is priority #1**

- Same AI categorization as Telegram
- Track influencers across both platforms
- Cross-platform synthesis (Telegram + Twitter in one digest)
- Thread summarization
- Crypto Twitter is where narratives form - we need to be there

**Why Now:** Crypto alpha happens on Twitter publicly, Telegram privately. Waiting is not an option. You need both platforms unified.

#### Project & Token Tracking (Q2-Q3 2026)
**The Feature:** Track specific projects, not just categories

- **Automatic extraction** - AI identifies tokens and project names in messages
- **Subject clustering** - All messages about same project grouped together
- **Dedicated project views** - See everything about a project in one place
- **Custom following** - Add any project to track, beyond our 6 categories

**Use Case:** You hold 10 tokens. Follow them all. Get unified updates on each, not scattered across categories.

**Add Sentiment Later:** Once tracking is stable, layer in bullish/bearish signals per project.

#### Community Features (Q3 2026)
**Leverage collective intelligence**

- **Shared channel lists** - Curated collections by topic/strategy
- **Collaborative filtering** - Community helps identify quality sources
- **Channel leaderboards** - See which channels provide most value
- **Import/export** - Share your setup with others

**Why:** Network effects. Users helping users find signal. This builds moat.

#### Smarter Filtering
**Better noise detection:**

- Channel quality signals (engagement patterns, community trust)
- Advanced spam recognition
- Continuous learning from new patterns

#### Discovery Engine
**Find better sources:**

- Personalized channel recommendations
- Similar channel suggestions
- Trending channels by category
- Quality indicators

#### More Control
**Customize your experience:**

- Choose digest delivery times
- Set summary length (concise/detailed)
- Mix categories that matter to you
